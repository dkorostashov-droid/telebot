LC Waikiki HR Telegram Bot

1️⃣ Встанови Python 3.10+
2️⃣ Створи папку LC_WAIKIKI_UA_HR_bot
3️⃣ Скопіюй у неї всі файли
4️⃣ Поклади credentials.json у цю саму папку
5️⃣ Встанови бібліотеки:
   pip install -r requirements.txt
6️⃣ Відкрий config.py і встав свій BOT_TOKEN
7️⃣ Запусти:
   python main.py

Бот автоматично:
- Збирає дані кандидатів (ПІБ, телефон, місто, ТРЦ)
- Зберігає у Google Sheets (LCWAIKIKI_candidates / work)
- Надсилає повідомлення в HR-канал
